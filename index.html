📘 このツールの使い方
1) 🧩 編成（タップで編集）
  • 第1〜第3軍（各5枠）＋控え（10枠）を、編成タブのタイルからタップして編集します。
  • 各スロットで キャラ と 武装Lv（0〜30） を設定できます。
  • 10人以上配置すると、🥇 最優先育成まとめ と 🚀 育成優先ランキング が自動更新されます。
  • 「🥇/🥈/🥉/🧰」ボタンで各軍や控えへジャンプできます。
2) 🛡️ 装備タブの使い方
  • 火力役 / 盾役 / 支援役 を切り替えると、その役割のキャラだけを装備比較できます。
  • 各キャラの 装備Lv と、表示される2種類の装備の ★ を入力すると、未完成度に応じて並び替えます。
  • 下の ★上げ判定（2キャラ比較） では、次にどの装備を★+1するのが良いかを簡易表示します。
3) 🚀 育成優先ランキングの見方
  • ランキングは、次の到達目標（Lv10 / 20 / 30）まで上げる時の効率をもとに並びます。
  • 弱点補強 / コスパ / 将来性 / 安全 のラベルは、その候補を今上げる理由の目安です。
  • 短い理由文は、いまの編成に対してどこが噛み合っているかを一言で補足しています。
4) 💾 データ保存について
  • 入力内容はブラウザの localStorage に自動保存されます（同じ端末・同じブラウザ内）。
  • 別端末や別ブラウザには引き継がれません。
